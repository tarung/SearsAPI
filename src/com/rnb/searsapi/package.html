This package provides Interfaces to search products and parts, stores, deals and local inventory in Sears, Kmart and Sears Marketplace

1. The Product Search API: Search API Interface to find products based on keyword and filtered by other parameters, and to fetch product category information.
2. TopSellers Search API: The Top Sellers API allows developers to retrieve the top-selling or most-viewed products from a given category within the Sears or Kmart catalog.
3. Deals Search API: Use the Deals API to retrieve Daily or Weekly specials for products and for specific stores.
4. Stores Search API: Fetch store details, including hours, location and more.
